# indecision-app

Simple react app to decide on something if you're confused.

# Install : 
Clone/Download the app
```
git clone
```
**cd** into the app
```
cd indecision-app
```
Install the dependencies using **yarn** or **npm**
```
yarn / npm install (depending on your package manager)
```
Start the app.
```
yarn start / npm start
```
